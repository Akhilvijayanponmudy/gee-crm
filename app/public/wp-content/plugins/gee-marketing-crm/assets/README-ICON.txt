WordPress.org Plugin Icon Requirements
=====================================

For your plugin to display the logo in the WordPress.org repository, you need to add an icon file in one of these locations:

1. **Recommended:** assets/icon-256x256.png
   - Size: 256x256 pixels
   - Format: PNG
   - Location: gee-marketing-crm/assets/icon-256x256.png

2. **Alternative:** assets/icon.svg
   - Format: SVG (scalable vector)
   - Location: gee-marketing-crm/assets/icon.svg

3. **Root directory options:**
   - icon-256x256.png (in plugin root)
   - icon.svg (in plugin root)

**Requirements:**
- PNG: Must be exactly 256x256 pixels
- SVG: Should be optimized and properly formatted
- The icon should represent your plugin clearly
- Use your logo image (the one with the envelope in shopping cart)

**How to add:**
1. Save your logo as a 256x256 PNG file
2. Name it: icon-256x256.png
3. Place it in: gee-marketing-crm/assets/icon-256x256.png

WordPress.org will automatically detect and use this icon for your plugin listing.
